Платформа и формат

Технология
Критичный выбор. iOS + Android нативно — дороже вдвое. React Native / Flutter — один код для обеих платформ, но есть ограничения. PWA — веб-приложение в браузере, быстрее и дешевле всего. Ваш выбор прямо влияет на стоимость.

Например, личный кабинет в браузере для тех, кто не хочет скачивать приложение.

Финансы

MVP — минимальный рабочий продукт для проверки идеи. Полный продукт — сразу весь заявленный функционал.
